*yt-dlp* is a command line video/audio downloader.

```shell
> yt-dlp <url> # download
> yt-dlp --write-thumbnail <url> # and download
> yt-dlp --audio-quality 5 # 0 for best, 10 for worst
> yt-dlp --remux-video mp4 # recode
> yt-dlp --embed-chaptres # for youtube
```
