level 1 runs on *bare metal*. A level-1 hypervisor is a minimal operating system with the hyprevisor software on top. Its pro is that it requires little resources for the host OS.

The most popular options are:
- [proxmox](proxmox)
- [xcp-ng](xcp-ng)
- [esxi](esxi)

Some NAS systems (like TrueNas Scale) also support running VMs.
