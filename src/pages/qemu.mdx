*Qemu* can be either used [directly via the CLI](/qemu-cli) if you need its advanced features or with a frontend:

| frontend         | host os | ui type | good for                      |
| ---------------- | ------- | ------- | ----------------------------- |
| utm              | macos   | gui     | running VMs on apple silicon  |
| [quickemu](/quickemu)     | linux   | cli     | automatically configuring VMs |
| [virt-manager](/virt-manager) | linux   | gui     | creating complex setups       |
