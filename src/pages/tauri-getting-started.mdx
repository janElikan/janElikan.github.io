Install the [system deps](https://v2.tauri.app/start/prerequisites/#system-dependencies). If you're on [nixos](/nixos), they have [a flake example](https://tauri.app/v1/guides/getting-started/prerequisites/#1-system-dependencies), which I have tweaked and will post on 2024-08-10 as a github gist.

```shell
> npm create tauri-app@latest -- --rc
> cd <name>
> npm install
> npm run tauri android init
> npm run tauri dev
# or npm run tauri android dev
```
